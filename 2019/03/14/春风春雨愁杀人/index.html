春风春雨愁杀人
发布于 2019-03-14   春天 闲扯 热度 0 喜欢 0

春天来了。

春天来了,冬天就要走了。冬天留在了去年,变成了去年的冬天。新的春天来了,旧的冬天死去了。

每个人都有一个季节。有的人的冬天没能变成春天,它死去了,变成了冷天灰蒙蒙的雨中遗落在路边的一块烂木头,它也不会再发芽了,它死了。它会零落成泥碾作尘,但是也没有香了。旧的泥上会开出新的花朵,旧的烂木头上会长出新的霉斑、新的青苔,可是它死了,活不过来了。

有的人的冬天活过来了,挺过来了,冬天挺过了冬天。所谓花花世界,所谓沉舟侧畔千帆过,它还会继续变化,变成一溜西瓜、一溜葡萄、一溜冬枣。我喜欢吃冬枣。

做好的菜要趁热吃,放久了会咸。苹果削好了半小时就要吃掉,鲜奶一整天就要喝掉。烟囱里头冒烟是正做饭,拉开门闸,灶上摇一摇铃,就得麻溜的去洗手端菜,这叫规矩,叫道理风暖了,鸟唱了,就差一场雨春天就到了。

半夜肚子饿去冰箱里觅食,冷气呼啦啦冲了出来,像冬天的冷风啪嗒啪嗒地抽打着门帘。冷气吹过开了没喝完的半盒牛奶,吹过窗台边高高低低的碗碟和花花绿绿的玻璃瓶,吹过客厅无人照料半死不活的绿萝、客房叠的整整齐齐沉静不语的羽绒被,和深夜里洗衣机的阵阵暄嚣。冷气吹到了窗户外头,吹的绿叶儿簌簌,直向上冲。向上飞去向上飞去，便看到许多星星，都成为亮晶晶的眼睛了。

天哗哗的下雨了。春天到了。

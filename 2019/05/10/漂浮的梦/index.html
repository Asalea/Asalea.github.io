漂浮的梦
发布于 2019-05-10   更新于 2019-05-26 冬天 漫想 热度 0 喜欢 0

​ 这是去年冬天的文了，那时候还跟小王在一块，看看还挺有意思。每每以为自己经历的已经蛮多的了，看的也蛮懂的了，还是会不知不觉的陷进去。看来玩玩吧。

​ 这里更像是一个树洞（学来的词），每次的心情也都是像在写一封不会寄出的情书，睡前把它压在枕下，却又偷偷摸摸的露出一角――――怕被发现，却又怕不被人发现。这是一种很隐秘的心情，渴望得到认同又觉得得不到认同。事实上很多时候也是这样的。
当然，不止是我，还有身边的朋友、亲人，还有许许多多陌生人。
但是还是会一得空就写一点，因为老觉得文字有点直指人心的意思，也算抱持着一点信念。

​ 之前被人指责“自己的世界里自己最高，把自己放在人生的第一位”，虽然不觉得哪里错误，但也觉得自己隐隐是有点问题；再深思，也发现还有许多一直以来的、发觉却从未深思过的伤害，还有许许多多未发觉所以可以安之若素的关心巴拉巴拉之类，一时间也是让我千头万绪。
不是想讲的含混，或者有什么隐秘，只是真的是一瞬尘世一瞬天堂，人事的转折太快，你还没看清就已经明明白白稀里糊涂的发生了，你只能在往后长长的日子里面慢慢的品咂。

​ 很久以来一直觉得自己想的通透、看的深远，秉持着一点似是而非的道家的态度：不主动、不强求、不执着，相逢开口笑，一笑泯恩仇，只向直中取，过后不思量。
这样的心态很舒服，很通达。你可以开开心心的去爱，开开心心的分开，潇洒的走进别人的人生，潇洒的离开。或者换句话说，在一个艳阳天，挎起包，套上门口衣架的大衣，轻轻开锁，轻轻掩门，却再无归期。

​ 一直这样过，顺风顺水倒也没什么差错，一切好或不好，我受之坦然也未深思。

​ 但其实仔细想一想，这里面是不是有一点不公平。我们去逛商场尚且有“请勿触摸”标志，即使是宽松如网购，剪标后想退换也不见得容易，但我们在人事、在更加虚无缥缈却更加值当珍贵的感情――我谈的是亲情、恋情、友情……――方面，为什么却没有一个明确的界限呢？
以下省略一万字。

​ 所以，活得太坦然的人，一定有人接过了你不快甚至悲伤的火把，噼啪噼啪、明明暗暗的燃烧着。
当你，夜夜酣睡的时候，可能就有个人，几年闻不得女人香。

​ 所有以上，起源于我昨天的一个梦。

​ 梦是一面镜子，它是漂浮在尘世上面的神秘，我们不用太追究，但是要相信。镜子对面，我想是另外一个我，跟我相似，但又受到光的折射反射散射各种射而有点不太一样的我，一个关于我没有真品的的赝品，我想是他在提醒我。

​ 一定有一个这样的人，不接受反驳。不然无法解释，为什么独处的时候也有被拥抱被陪伴的感觉。
